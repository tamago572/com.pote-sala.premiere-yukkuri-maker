# 開発用メモ

## ディレクトリ構成

```text
|---- dist/
|       |---- build時に生成されるファイル。githubには上げない。
|
|---- docs/
|       |---- ここ
|
|---- src/
        |---- js/
        |       |
        |       |---- main/ (Mainコンポーネント 開いたとき最初に表示されるやつ。localhost:3000/main/index.html)
        |       |        |---- index.html (これは編集しない)
        |       |        |---- index.tsx  (これも編集しない)
        |       |        |---- main.tsx (コンポーネント。こういったものを編集する)
        |       |
        |       |---- components/
        |               |---- SettingsArea.tsx
        |
        |
        |---- jsx/
                # ExtendScriptをこのディレクトリ配下に配置

                ppro/
                    ppro.ts (evalES, evalTS で参照されるファイル。実行したい関数が他のファイルに記述してある場合、このファイルに import, from する)

                utils/
                    util.ts (共通で使用する関数。使用したい場合、ppro.tsでimportしないといけない)
...
|
| package.json、PrettierやESLintなど
```
